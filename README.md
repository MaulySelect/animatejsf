AnimateJSF
=========

A thin JSF Library based on [animate.css]

Version
----

1.0

Demo
-----------
[animatejsf.org]

Installation
--------------

```sh
<dependency>
       <groupId>org.animatejsf</groupId>
       <artifactId>animatejsf</artifactId>
       <version>1.0-SNAPSHOT</version>
</dependency>
```

Usage
--------------
```sh
<aj:animate target="formId" type="tada" />
```
